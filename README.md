# Pizza_Sales_Analysis

# 1. Project Overview 
A SQL-based analytics project performed on a pizza sales dataset to provides insights into orders, revenue, product mix, and customer behaviour. This project helps understand sales performance, customer ordering patterns, and revenue drivers in a pizza business.

# 2. Objectives
* Understand key performance metrics of pizza sales
* Analyze customer ordering habits
* Identify top-performing pizza categories and types
* Perform advanced revenue-based analytics
* Improve query-writing skills from beginner to advanced level

# 3. Dataset Description 
* orders – Order-level information 
* order_details – Line-level details 
* pizzas – Price and size details
* pizza_types – Category and pizza names

# 4. Business Questions 
* Retrieve the total number of orders placed.
* Calculate the total revenue generated from pizza sales.
* Identify the highest-priced pizza.
* Identify the most common pizza size ordered.
* Join the necessary tables to find the total quantity of each pizza category ordered.
* Join relevant tables to find the category-wise distribution of pizzas.
* Determine the top 3 most ordered pizza types based on revenue.
* Calculate the percentage contribution of each pizza type to total revenue.
* Analyze the cumulative revenue generated over time.
* Determine the top 3 most ordered pizza types based on revenue for each pizza category.

# 5. Tools Used

* MySQL Workbench for SQL queries
* GitHub for project documentation

# 6.Key Insights You Can Derive
* Best-selling pizzas and categories
* Customer ordering time patterns
* Revenue drivers
* Product contribution analysis
* Daily sales trend and cumulative performance
